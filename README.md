# Build a Spring Boot application with GORM

This repository contains the worked sample app for the [Build a Spring Boot application with GORM](https://grails.apache.org/guides/gorm-without-grails/4/guide/index.html) guide on the Apache Grails site.

## Run the example

`./gradlew bootRun`

## Maintained at apache/grails-static-website

The guide narrative is maintained at https://github.com/apache/grails-static-website. Changes to the guide content go there; this repo holds the worked sample app only.
